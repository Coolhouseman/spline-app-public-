Understanding the Buffer

  • When users deposit $100, BlinkPay charges 0.1% fee, so you receive $99.90
  • User's wallet shows $100, creating a $0.10 buffer requirement
  • Fast withdrawals (2% fee) help offset this - user deposits $100, withdraws fast, you keep $2
  • Normal withdrawals are free - if user withdraws $100, you need full $100 to send
